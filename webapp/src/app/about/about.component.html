Course: Open Linked Big Data

Team members: Jacqueline Neef and Moritz Meister

This is a project developed at Universidad Politécnica de Madrid for the course "Open Linked Big Data". We used open data published by the Ayuntamiento de Madrid on Museums and Monuments, preprocessed it, defined a resource naming strategy, built an ontology (reusing the existing ontologies schema.org and vcard) linked it to external data (wikipedia and Ayuntamiento de Madrid) and transformed it to RDF linked data. The result is a web application in which you can explore the monuments and museums of Madrid, navigate based on context and find additional information.
